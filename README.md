# Payment App (Learning Project)

## Description
This is a beginner-friendly backend project built to understand how real-world payment systems work.  
The project evolves step-by-step from Core Java → JDBC → Spring Boot.

## Tech Stack
- Core Java
- JDBC
- Spring Boot
- MySQL

## Project Evolution
1. Core Java (basic logic)
2. JDBC (database integration)
3. Spring Boot (REST APIs)

## Future Improvements
- Authentication & security
- Payment gateway integration
- UI (frontend)
